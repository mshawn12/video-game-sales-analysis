# video-game-sales-analysis
Using a Python Flask-powered API, HTML/CSS, Javascript and SQL to test hypotheses about video game sales

## Background Information
The team will leverage a Global Video Game Sales & Ratings dataset from Kaggle (https://www.kaggle.com/datasets/thedevastator/global-video-game-sales-ratings) in order to test various hypothesis about video game genres.
